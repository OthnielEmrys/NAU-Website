NAU - Web Development

Web Development

Course Overview

The Web Development course at New Age University covers everything you need to build dynamic, responsive websites from scratch. You will learn front-end development with HTML, CSS, JavaScript, and back-end development using popular frameworks and databases. This course is designed for beginners and intermediate learners who want to pursue a career in web development or enhance their coding skills for personal projects.

Key Details

  • Duration: 10 weeks
  • Level: Beginner to Intermediate
  • Format: Online, Self-paced
  • Start Date: October 1, 2024
  • Cost: Free

Curriculum

The curriculum for this course includes the following modules:

  • Week 1: Introduction to Web Development - Understanding how websites work and the tools you will need.
  • Week 2: HTML & CSS Basics - Building the structure and styling of web pages.
  • Week 3: JavaScript Basics - Adding interactivity to websites.
  • Week 4: Advanced JavaScript - Working with DOM manipulation and asynchronous programming.
  • Week 5: Front-End Frameworks - Introduction to Bootstrap, React, or Vue.js for rapid development.
  • Week 6: Version Control with Git - Collaborating and managing code with GitHub.
  • Week 7: Introduction to Back-End Development - Working with Node.js, Express.js, or similar frameworks.
  • Week 8: Database Management - Introduction to SQL and NoSQL databases.
  • Week 9: Deploying Websites - Hosting and managing your websites live.
  • Week 10: Full-Stack Project - Developing a full-stack web application.
